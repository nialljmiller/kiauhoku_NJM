rocrit
=================

Identical to those presented by `Claytor et al. (2020) <https://ui.adsabs.harvard.edu/abs/2020ApJ...888...43C/abstract>`_, but uses the stalled magnetic braking law of `van Saders et al. (2016) <https://ui.adsabs.harvard.edu/abs/2016Natur.529..181V/abstract>`_ with fast launch condition.

.. code-block:: python

    import kiauhoku as kh
    rocrit = kh.load_grid('rocrit')
