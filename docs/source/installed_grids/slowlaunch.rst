slowlaunch
=================

Presented by `Claytor et al. (2020) <https://ui.adsabs.harvard.edu/abs/2020ApJ...888...43C/abstract>`_, using the magnetic braking law of `van Saders et al. (2013) <https://ui.adsabs.harvard.edu/abs/2013ApJ...776...67V/abstract>`_ with slow launch condition.

.. code-block:: python

    import kiauhoku as kh
    slowlaunch = kh.load_grid('slowlaunch')
